# Vero
> Authentic Italian, refined for the modern palate.

## 🚀 Live Demo
https://vero-1777479354.vercel.app

## ✨ Features
- **Immersive Hero**: A cinematic, full-bleed video background capturing the artistry of traditional pasta preparation.
- **Dynamic Menu**: A high-fidelity, categorized menu interface featuring detailed ingredient origin stories.
- **Reservation Engine**: A seamless, real-time table booking flow designed for premium user convenience.
- **Editorial Narrative**: A dedicated Chef’s section highlighting the rich heritage and philosophy behind the kitchen.
- **Premium Animations**: Sophisticated scroll-triggered reveals and soft-focus typography transitions powered by Framer Motion.

## 🛠 Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## 🏗 Architecture
The project follows a modular architecture within the Next.js App Router, separating concerns into reusable UI components located in `/components` and page-specific logic within `/app`. This structure ensures a clean separation between the editorial narrative, menu data presentation, and the reservation booking flow, facilitating easy maintenance and scalability.
